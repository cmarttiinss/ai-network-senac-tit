1. Conectar ao MySQL como Root
sudo mysql -u root -p

Quando conectado, o prompt mudará para:
mysql>

2. Listar os Bancos de Dados
Dentro do prompt do MySQL, execute:
SHOW DATABASES;
A saída será parecida com:
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
| seu_banco_aqui     |
+--------------------+

3. Sair do MySQL
EXIT;
